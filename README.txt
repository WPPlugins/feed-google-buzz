== Description ==

Show your post in google buzz on your wordpress

More info <a href="http://blog.satiro.es/feed-google-buzz-google-buzz-para-wordpress/">blog.satiro.es</a>

== Installation ==

1. Upload 'feed-google-buzz' in the `/ wp-content/plugins /`
2. Activate the plugin from 'Plugins' menu in WordPress
3. Put your Gmail username (without '@ gmail.com') in the configuration page
4. Place the widget in the sidebar.
